记录hexo使用过程遇到的问题和技巧，方便以后自行查阅

1. hexo 修改生成.md 文件的头部

  • 使用sublime全文搜索后，查询到 blog目录下面的 node_modules\hexo\node_modules\hexo-cli\assets\scaffolds\ 文件，修改目录下面的.md文件，执行 bash hexo n "test" 发现，修改的没有生效。
    于是继续查找搜索结果，发现blog的目录下面也有个一个scaffolds\文件夹，修改这个目录下面的.md文件，再次执行
    bash hexo n "test"， 生效。

2. 使用hexo写文章注意

  • 对于位于标题下面的文章段落，需要在编写的时候从其实位置开始或者用空格移动内容，不能使用tab缩进开头，会解析为 pre标签
